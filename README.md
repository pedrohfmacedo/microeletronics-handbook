# Microelectronics Handbook

Technical handbook, glossary and reference guide focused on Microelectronics, VLSI, ASIC Design, Verification, Digital Design and Physical Design.

---

# Overview

This repository was created as a technical knowledge base for students, engineers and researchers working with:

* Microelectronics
* VLSI
* ASIC Design
* Digital Design
* Physical Design
* Verification
* DFT
* Timing Analysis
* Semiconductor Fundamentals

The main goal of this project is to organize technical concepts, abbreviations, files, flows, tools and definitions used in the semiconductor industry into a structured and easy-to-access format.

This repository works as:

* Mini technical dictionary
* VLSI glossary
* Study handbook
* Quick reference guide
* Knowledge base

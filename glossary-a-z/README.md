<a name="a-pt"></a>

<details markdown="1">
<summary><b>A</b></summary>

| Sigla           | Nome                                    | Descrição                                                                             |
| --------------- | --------------------------------------- | ------------------------------------------------------------------------------------- |
| *ABV*           | Assertion-Based Verification            | Método de verificação que usa assertions para validar comportamentos do circuito.     |
| *ASIC*          | Application-Specific Integrated Circuit | Circuito integrado desenvolvido para uma aplicação específica.                        |
| *ATPG*          | Automatic Test Pattern Generation       | Processo automático de geração de vetores de teste.                                   |
| *Aperture Time* | Aperture Time                           | Intervalo crítico em torno da borda do clock onde os dados devem permanecer estáveis. |

</details>

<a name="b-pt"></a>

<details markdown="1">
<summary><b>B</b></summary>

| Sigla           | Nome                          | Descrição                                                                |
| --------------- | ----------------------------- | ------------------------------------------------------------------------ |
| *BDD*           | Binary Decision Diagram       | Estrutura gráfica usada para representar funções booleanas.              |
| *BF16*          | bfloat16                      | Formato reduzido usado em IA, mantendo grande expoente e menor mantissa. |
| *BFM*           | Bus Functional Model          | Modelo usado para simular interfaces e barramentos.                      |
| *BILBO*         | Built-In Logic Block Observer | Estrutura usada para geração de padrões e análise de respostas em BIST.  |
| *BIST*          | Built-In Self-Test            | Técnica onde o próprio circuito realiza testes internos automaticamente. |
| *BJT*           | Bipolar Junction Transistor   | Transistor usado para amplificar ou chavear sinais elétricos.            |
| *Block Diagram* | Block Diagram                 | Representação gráfica de um sistema usando blocos funcionais.            |

</details>
